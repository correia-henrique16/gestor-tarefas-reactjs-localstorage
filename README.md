# Gestor de Despesas

Este projeto foi desenvolvido para praticar a implementação do react. Desta vez com uma aplicação funcional onde o utilizador pode controlar as suas finanças, definindo o valor da despesa e a categoria da mesma. 

Contém um filtro por categoria, e um cálculo do total do valor das despesas registadas. 



---

## Tecnologias utilizadas
- React.Js
- JavaScript
- CSS


## Funcionalidades
- Adicionar novas despesas
- Histórico de despesas com data e opção de eliminação
- Visualização do total gasto em despesas. 

## O que aprendi
- **LocalStorage:** Implementação do localStorage para o user manter o histórico de despesas mesmo depois de atualizar ou fechar o browser.
